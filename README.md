## Running Locally


```sh-session
git clone https://github.com/whoviks/whoviks_web.git
cd whoviks_web
```


Create a `.env` file similar to [`.env.example`](https://github.com/whoviks/whoviks_web/blob/main/.env.example).

Then install dependencies and run the development server:
```sh-session
npm install
npm run dev
```
